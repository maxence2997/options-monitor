## 二、各策略詳細規則

### 策略 1：SPY Iron Condor（定義風險、穩定收 premium）

**適用情境：** SPY 橫盤震盪時獲利，大行情時定額虧損

**進場規則（每月第一個交易日）：**
```
DTE        = 30-45 天（找下個月到期的週五）

Put Short  = 當日 SPY 現價 × 94%
Put Long   = 當日 SPY 現價 × 91%
Call Short = 當日 SPY 現價 × 106%
Call Long  = 當日 SPY 現價 × 109%

張數 = 20 張
```

**出場規則（系統自動通知）：**
```
✅ 獲利達 50% premium → 整組平倉，等下月
❌ 虧損達 200% premium → 整組平倉
   （例：收 $300，虧 $600 就出場）
⏰ DTE ≤ 7 天且有獲利 → 平倉，不冒 Gamma 風險
```

**最大虧損固定：** Spread 寬度（3%）× 100 × 20 張 - 收到 premium，上限可預期。

---

### 策略 2：NVDA Wheel（CSP → CC，高 premium 輪轉）

**適用情境：** 看好 NVDA 長期，願意在低點接股票，邊等邊收 premium

#### Phase A：Cash-Secured Put（CSP）

**進場規則：**
```
Strike = 當日 NVDA 現價 × 88%（留更多安全空間，因為 NVDA 波動大）
DTE    = 21-30 天（找對應的週五到期）
張數   = 15 張
```

**出場規則：**
```
✅ 獲利達 50% premium → 提前買回平倉，開下一輪
✅ 到期自然歸零（未被 Assign）→ 開下一輪 CSP
❌ 虧損達 300% premium → 平倉出場（這個月不再開新倉）
⚠️ 被 Assign（股票跌破 Strike）→ 進入 Phase B
```

#### Phase B：Covered Call（CC，接股票後）

**進場規則（被 Assign 後立即開）：**
```
Strike = 被 Assign 的成本價 × 105%
DTE    = 14-21 天
張數   = 15 張（同等股票數量：1500 股 = 15 張）
```

**出場規則：**
```
✅ 獲利達 50% premium → 提前買回，重開 CC
✅ 到期未被 Assign → 繼續賣下一輪 CC
✅ 被 Assign（股票被以 Strike 賣出）→ 回到 Phase A，開新 CSP
```

**Wheel 循環示意：**
```
開 CSP（收 premium）
    ↓
未被 Assign → 重複開 CSP ←──────────────────┐
    ↓                                         │
被 Assign（接股票）                           │
    ↓                                         │
開 CC（收 premium）                           │
    ↓                                         │
未被 Assign → 繼續開 CC                       │
    ↓                                         │
被 Assign（股票賣出）→ 回到起點 ──────────────┘
```

### 策略 3：QQQ Bull Call Spread（方向性押注）

**適用情境：** 看漲 QQQ，用固定成本賭上漲，最大虧損只有 premium 本金

**進場規則（每 2 個月 1 次）：**
```
買 10 張 QQQ Call（ATM 附近）
賣 10 張 QQQ Call（+10% Strike）
DTE = 45-60 天
premium 為負數（你付出的成本，例：-3.20）
```

**出場規則：**
```
✅ 獲利達 100%（成本翻倍）→ 平倉
❌ 虧損達 100%（premium 全部虧光）→ 平倉
   （最大虧損 = premium 本金，不會再多了）
```

---

### 策略 4：SPY Hedge Put（黑天鵝保險）

**適用情境：** 市場崩盤時對沖其他部位虧損，買了不用管

**進場規則（每季 1 次）：**
```
買 5 張 SPY Put
Strike = 當日 SPY 現價 × 85%（深度 OTM）
DTE    = 90 天左右
premium 為負數（付出的保費）
```

**持有邏輯：**
```
市場正常  → 這筆錢當保費，到期歸零沒關係
市場崩盤（跌幅 >15%）→ 這個 Put 大幅獲利，對沖其他部位虧損
```

不主動出場，到期歸零或大漲才考慮賣出。

## 四、競賽策略思維

### 為什麼這樣配置？

這是**競賽**（需要贏過其他人）而非**真實投資**（只要正報酬）：

| | 真實投資思維 | 競賽思維 |
|---|---|---|
| 目標 | 保本 + 穩定增長 | 跑贏其他參賽者 |
| 風險態度 | 保守 | 可以承擔更多 |
| 最壞情況 | 不能虧錢 | 虧了沒關係（假錢） |
| 時間 | 無限 | 死線 9 月中 |

### 預期報酬（6 個月）

| 情境 | 預期報酬 | 說明 |
|------|---------|------|
| 樂觀 | +35% | 市場橫盤、QQQ 上漲 |
| 基本 | +15% | 市場正常波動 |
| 保守 | +8% | 部分策略停損 |
| 最差 | -10%（估） | 市場單邊大行情 |

## 七、收到通知後的處理 SOP

| 通知 | 立即動作 |
|------|---------|
| 🎯 獲利達標 | Moomoo 買回平倉 → 開下一輪 |
| 🛑 停損觸發 | Moomoo 市價買回 → 這個月不開新倉 |
| ⏰ DTE ≤ 7 天 | 有獲利就平倉；OTM 快到期可讓其歸零 |
| ⚠️ Assignment 風險 | 確認 Moomoo 現金充足，或考慮 Roll 展期 |
| ⚠️ IC 翼突破 | 評估是否整組平倉，或單獨平掉突破的那側 Spread |
| 📊 每日收盤總結 | 看一眼確認無異常，不用操作 |

### Roll（展期）操作說明

當期權快到期但想繼續持有部位時：
```
1. 在 Moomoo 買回當前合約（平倉）
2. 賣出新合約（相同或調整的 Strike，更遠的到期日）
```

## 八、期權概念快速複習

### 賣方 vs 買方

| | 賣方（本策略主力） | 買方 |
|---|---|---|
| 付/收 premium | 收 premium | 付 premium |
| 時間方向 | 時間是朋友（Theta 正） | 時間是敵人 |
| 勝率 | 高（但每次賺少） | 低（但大勝時賺多） |
| 風險 | 需要保證金/現金 | 最多虧 premium |

### P&L 計算

**賣方策略（CSP / CC / IC）：**
```
開倉收 premium = $3.50/股 × 100 × 15 張 = $5,250
若現在只需 $1.75 買回 → 已獲利 50% → 建議平倉
P&L % = (3.50 - 1.75) / 3.50 × 100 = 50%
```

**買方策略（Bull Call Spread / Hedge Put）：**
```
開倉付 premium = $3.20/股 × 100 × 10 張 = $3,200（成本）
若現在可以 $6.40 賣出 → 獲利 100% → 建議平倉
P&L % = (6.40 - 3.20) / 3.20 × 100 = 100%
```

### Iron Condor 結構

```
Put Long ──── Put Short ──── [安全區間] ──── Call Short ──── Call Long
  91%              94%                            106%              109%

股價在安全區間內到期 → 全部 premium 歸你
股價跌破 Put Short   → 開始虧損，最多虧到 Put Long 為止
股價漲過 Call Short  → 開始虧損，最多虧到 Call Long 為止
```
